# Smenka Mobile

Flutter-приложение для учёта рабочего времени. Два режима работы:

- **Персональный** — трекер смен для себя с таймером, паузами и историей
- **Организационный** — управление сотрудниками, геопроверка, статистика по сменам

## Стек

Flutter, Dart, flutter_bloc / Cubit, auto_route, freezed, Dio, Firebase (Crashlytics, Remote Config), Yandex Maps
